# Images Directory

Place your authentication page images here:

## Required Images

1. **login-hero.jpg** (or .png)
   - Smart agriculture scene with farmers using technology
   - Golden hour lighting
   - Blue/green/gold color scheme
   - Recommended size: 1200x1200px or larger

2. **signup-hero.jpg** (or .png)
   - Futuristic agricultural innovation scene
   - Purple/indigo/pink gradient colors
   - Tech tree or growth imagery
   - Recommended size: 1200x1200px or larger

## Image Specifications

- **Format:** JPG or PNG
- **Aspect Ratio:** 1:1 (square) works best
- **Resolution:** Minimum 1200x1200px (2000x2000px recommended)
- **File Size:** Optimize to under 500KB for faster loading
- **Naming:** Use exact filenames: `login-hero.jpg` and `signup-hero.jpg`

## How to Add Images

1. Save your generated images to this directory
2. Name them exactly as: `login-hero.jpg` and `signup-hero.jpg`
3. The images will automatically appear on the Login and Signup pages
4. If images fail to load, the pages will gracefully fall back to gradient backgrounds

## Tips

- Use image optimization tools to reduce file size
- Test images on different screen sizes
- Ensure images are high quality but optimized for web
- Consider using WebP format for better compression (update code if using WebP)
